###MeMeMe###


========================================================================
DESCRIPTION:
This app enables a user to take a picture, and add text at the top and bottom to form a meme, and share that meme on Facebook and Twitter and also by SMS or email. In this project, sent memes will appear in a tab view with two tabs: a table view and a collection view.


Supported Formats


=========================================================================
BUILDS REQUIREMENTS:


Xcode 8.2.1,  IOS SDK 8.2 or better, 


=========================================================================

iPhone, iPad, or iPod Touch running iOS 8.2.1 or better Xcode 8.2.1,  IOS SDK 8.2.1 or better,

PACKAGING LIST:


MemeTableViewController.swift
A UIViewController provides a root view controller to be the start point of the app and populates all the meme records in table view. 


MemeEditorViewController.swift
A UIViewController implements to able to update the selected meme record.


MemeCollectionViewController.swift
MemeCollectionViewController populates all the meme records in collection view. 


MemedDetailViewController.swift
MemedDetailViewControlle populates all the meme object. 


MemeCollectionViewCell.swift
MemeCollectionViewCell
 class conforms UICollectionViewCell to customize the cell output for each meme object.  


MemeDataStructure.swift
Define a data type structure for Meme.
